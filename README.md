# 🔗 Shortalytics

**Shortalytics** is a full-stack URL shortener web application that not only helps you shorten long URLs, but also provides powerful **click analytics** and a sleek **dashboard** to track performance over time.

---

## 🌟 Key Features

- 🔐 **JWT-Based User Authentication**  
- ✂️ **Shorten Long URLs** with unique or custom codes  
- 📊 **Analytics Dashboard** showing:
  - Total Clicks
  - Click Trends Over Time
  - Timestamp, Device Info  
- 📁 **Link Management** with created & last accessed times  
- 📈 **Graphical Insight** with Chart.js  
- ⚙️ **RESTful API Support** for developers  
- 🎨 **Modern UI** using Tailwind CSS & Framer Motion

---

## 🧠 Why "Shortalytics"?

> “Shortalytics” = **Shorten** + **Analytics**  
> A unique name that captures the dual functionality of the platform: shortening URLs and providing rich analytical insights.

---

## 🖥️ Tech Stack

### 🚀 Frontend
- [React.js](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [React Query](https://tanstack.com/query/latest)
- [Chart.js](https://www.chartjs.org/) via `react-chartjs-2`
- Axios, React Icons, React Router

### 🔧 Backend
- [Spring Boot](https://spring.io/projects/spring-boot)
- [Spring Security](https://spring.io/projects/spring-security)
- [JWT Authentication](https://jwt.io/)
- [MySQL](https://www.mysql.com/)
- RESTful APIs with layered architecture (Controller, Service, Repository)
